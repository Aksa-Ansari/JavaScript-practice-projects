#  Spotlight Cursor Effect (HTML + CSS + JavaScript)

This project creates a spotlight-style lighting effect that follows the user‚Äôs mouse cursor.  
As the cursor moves a radial gradient highlights the area under the pointer, just like a moving flashlight in a dark environment.

## Features

- Smooth spotlight/flashlight effect
- Follows mouse movement in real-time
- Uses CSS variables for dynamic updates
- Beautiful radial gradient for soft lighting
- Clean and minimal code structure

## Technologies Used

- HTML
  - Structure of the spotlight container
- CSS3
  - Radial Gradient for spotlight effect  
  - CSS Variables (`--x`, `--y`) to track cursor position  
  - Background image styling
- JavaScript 
  - `mousemove` event listener  
  - Dynamic updates using `style.setProperty()`  
  - Real-time interaction with CSS variables
