# Text Decode Effect

A simple hover-based text animation where the text gets scrambled and then reveals the original message.

## Features
- Hover to start animation  
- Random letter scrambling  
- Smooth decode reveal   

## How It Works
When the user hovers over the text, each character changes into random letters and then slowly returns back to the original message.

## Technologies Used
- HTML  
- CSS  
- JavaScript
